Image Set / 8 Plates

A variable visual record.

This project carries 8 image plates for now. The first plate holds the major image, and the following plates act as replaceable placeholders for final drawings, renders, site photos, process frames, or material studies.

Reading

What this page is trying to hold.

The work is presented as a time-based spatial emotion, not a static image set. Anantam keeps its focus on time, routine, and repetition, while the composition leaves room for drawings, site notes, and expanded documentation.
